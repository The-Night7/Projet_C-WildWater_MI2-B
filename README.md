# Projet_C-WildWater_MI2-B
Le projet C-WildWater consiste à développer une appli composée d’un script Shell et d’un programme en C pour analyser et traiter un grand volume de données correspondant à 1/3 de la distribution d’eau potable en France. L’objectif est de réaliser des calculs sur un fichier CSV massif et de produire des résultats exploitables sous forme de fichiers et graphiques.

## Présentation
C-WildWater est un projet pour analyser et visualiser les données liées à la gestion de l'eau.
