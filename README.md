# Projet_C-WildWater_MI2-B
Le projet C-WildWater consiste à développer une appli composée d’un script Shell et d’un programme en C pour analyser et traiter un grand volume de données liées à la distribution d’eau potable en France. L’objectif est de réaliser des calculs sur un fichier CSV massif et de produire des résultats exploitables sous forme de fichiers et graphiques.
